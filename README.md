# Atv_CORS_PWII_MariaJulia2DS
Atividade para testar o CORS 

# Como rodar a aplicação :
- No terminal entre na pasta do projeto: cd Atv_CORS_PWII_MariaJulia2DS
- Em seguida, entre na pasta do backend: cd backend , instale as dependências com: npm i e inicie o servidor com : node server.js
- Depois, entre na pasta do frontend: cd frontend,  instale as dependências com: npm i e inicie o frontend na porta 3000 com : serve -l 3000 (obs: se o comando serve não for reconhecido pelo seu terminal, instale-o globalmente com : npm install -g serve )
- Para testar o cors funcionando, no seu navegador acesse: http://localhost:3000
